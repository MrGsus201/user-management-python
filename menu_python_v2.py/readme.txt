# User Management System in Python (v2.0)

This project is a user management system built in Python.
It allows adding, searching, editing, and deleting users through an interactive console menu.

This version (v2.0) introduces data persistence using JSON files, making the system more realistic and closer to a real backend structure.

This is a beginner practice project created for learning and educational purposes.

---

## Features

* Add new users
* Search users by ID or name/last name
* Edit existing user information
* Delete users with confirmation
* Display the full database
* Persistent data storage using JSON

---

## How to run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the main file:

python Menu_Python.py


---

## Project Structure

* `Menu_Python.py` → Main program
* `datos.json` → User database (persistent storage)

---

## What I learned

* Working with JSON files (`json.load` / `json.dump`)
* Data persistence (saving data outside RAM)
* Error handling with `try/except`
* Structuring code with functions
* Basic CRUD operations (Create, Read, Update, Delete)

---

## Future Improvements

* Refactor code into modules
* Improve ID system (UUID or better handling)
* Implement case-insensitive and partial search
* Add input validation (email, phone, etc.)
* Build a graphical user interface (GUI)
* Connect to a real database (SQLite, PostgreSQL)

# Sistema de Gestión de Usuarios en Python (v2.0)

Este proyecto es un sistema de gestión de usuarios hecho en Python.
Permite agregar, buscar, editar y eliminar usuarios utilizando un menú interactivo en consola.

Esta versión (v2.0) introduce persistencia de datos mediante archivos JSON, haciendo el sistema más cercano a un backend real.

Este es un proyecto de práctica creado con fines educativos y de aprendizaje.

---

## Características

* Agregar nuevos usuarios
* Buscar usuarios por ID o nombre/apellido
* Editar información existente
* Eliminar usuarios con confirmación
* Mostrar la base de datos completa
* Persistencia de datos usando JSON

---

## Cómo ejecutar

1. Asegúrate de tener Python instalado
2. Descarga o clona este repositorio
3. Ejecuta el archivo principal:


python Menu_Python.py

---

## Estructura del proyecto

* `Menu_Python.py` → Programa principal
* `datos.json` → Base de datos de usuarios (persistente)

---

## Lo que aprendí
